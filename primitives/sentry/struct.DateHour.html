Expand description

DateHour holds the date and hour (only).

It uses chrono::DateTime when serializing and deserializing. When serializing it always sets minutes and seconds to 0 (zero). When deserializing the minutes and seconds should always be set to 0 (zero), otherwise an error will be returned.
